# C_PROJECT

A collection of classic C programming projects demonstrating core concepts of the C language, including basic algorithms, control structures, file handling, and interactive applications.  
This repository contains multiple small programs that can help beginners explore practical implementations of C fundamentals.

---

## 📌 Project Overview

This repository includes the following C programs:

- **Calculator** – Basic arithmetic operations  
- **Complex Calculator** – Extended operations including complex numbers  
- **Snake Game** – Console-based snake game  
- **Tic-Tac-Toe** – Two-player game implementation  
- **Other utility programs** – Various C language exercises

Each program showcases hands-on usage of functions, loops, arrays, and user interaction in C.

## 🛠 Prerequisites

To build and run these programs locally, you’ll need:

- A C compiler (such as **GCC** or **Clang**)  
- A terminal or console environment (Windows, Linux, or macOS)

---

## 🚀 How to Compile & Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Adarshthakur-850/C_PROJECT.git
````

2. Navigate into the folder:

   ```bash
   cd C_PROJECT
   ```

3. Compile a C file (example with `calculator.c`):

   ```bash
   gcc calculator.c -o calculator
   ```

4. Run the compiled program:

   ```bash
   ./calculator
   ```
